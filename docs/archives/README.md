# Archived Patching Documentation

This folder contains the original detailed guides that were used during development of the patching system. These have been consolidated into the comprehensive **[PATCHING_GUIDE.md](../PATCHING_GUIDE.md)**.

## Archived Documents

- **AI_PATCH_UTILS_GUIDE.md** - Original guide for the AI patch utilities module
- **TEXT_OPERATIONS_GUIDE.md** - Detailed guide for text operations extension
- **PATCHING_BEYOND_JSON.md** - Guide for non-JSON type support
- **PATCH_EXTENSIONS_SUMMARY.md** - Summary of all patch extensions

## Why Archived?

These documents contain valuable detailed information but have been superseded by the single comprehensive guide. They're kept here for:

1. **Historical reference** - Understanding the development process
2. **Detailed examples** - Additional examples not in the main guide
3. **Implementation details** - Deep technical details for contributors
