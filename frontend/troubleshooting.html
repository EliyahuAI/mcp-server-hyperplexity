🔧 Troubleshooting Guide

Common solutions for data processing issues

🟥 Processing Issues

Common Problem: File Processing Errors

If you're experiencing issues during preview or validation processing, the most common cause is related to file format or table structure.

✅ Solution Steps (Try in this order):

1. Use Basic Table Format

  • Ensure your data is in a simple table format
  • Column headers should be in the first row only
  • Remove any extra rows above or below your data
  • Use only one table per file - no multiple tables
  • Avoid merged cells, complex formatting, or embedded charts

2. Convert to CSV Format

  • Save your file as CSV (.csv) instead of Excel (.xlsx)
  • CSV format eliminates complex formatting issues
  • In Excel: File → Save As → CSV (Comma delimited)
  • In Google Sheets: File → Download → Comma-separated values (.csv)

3. Check for Unusual Characters

  • Remove or replace special characters in your data
  • Check for: émojis, smart quotes (""), unusual symbols (™, ®, ©)
  • Replace with standard alternatives: regular quotes (""), (TM), (R), (C)
  • Avoid characters from other languages if not necessary
  • Use standard ASCII characters when possible

⚠️ What to Avoid

  • Multiple worksheets: Use only one sheet with your data
  • Headers in multiple rows: Keep headers in row 1 only
  • Empty rows/columns: Remove blank rows at the top or between data
  • Formulas: Replace formulas with their calculated values
  • Large files: If over 10MB, consider splitting into smaller files

📋 File Format Checklist

Before uploading, verify your file has:

  • ✓ Column headers in the first row
  • ✓ Data starts immediately after headers (no empty rows)
  • ✓ Consistent data types in each column
  • ✓ No merged cells or complex formatting
  • ✓ Standard characters (avoid émojis/symbols)
  • ✓ Single table structure
  • ✓ File size under 10MB

🎯 Still Having Issues?

If problems persist after trying the above steps:

  1. Try a smaller sample: Test with just 10-20 rows of your data
  2. Create a new file: Copy your data to a completely new Excel/CSV file
  3. Check your internet: Ensure you have a stable connection
  4. Try a different browser: Sometimes browser-specific issues occur

🔍 Advanced Troubleshooting

If you've tried all the above steps and still experience issues, this may indicate a more complex problem with your data structure or our processing system. Our technical team will investigate and contact you with a solution.
