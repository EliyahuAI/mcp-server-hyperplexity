# Gemini Development Notes

## Infrastructure

For a complete overview of the project's architecture, infrastructure, and setup, please refer to the [Infrastructure Guide](docs/INFRASTRUCTURE_GUIDE.md).

## Development Environment

Please note the following limitations of the development environment:

*   **Environment**: The development environment is a Windows machine running Ubuntu on WSL.
*   **Character Encoding**: Do not use non-standard Unicode characters in testing scripts or any other files, as this can cause issues in the specified environment. Stick to standard ASCII or UTF-8 where possible.
